Crowdsourced Brain Mapping

--------------------------

There are lots of ways to map the brain. This database puts a crowdsourced layer of metadata onto research papers. The goal is to help neuroscientists share and find relevant data that use different mapping methodologies or terminologies (nomenclatures).

To help out, take a look at these neuroscience publications. When you find a paper you might want to add, make sure to right click and "Copy Link Address" to paste it into the URL field, or else you won't be able to get the URL easily.

  • Proceedings of the National Academy of Sciences (Neuroscience)
  • Nature Neuroscience
  • Neuron, Cell Press

Upcoming Features:

  • "Add Your Own Feature", we are planning to update the user design so that it is easy and intuitive to both add and create new feature types. For example, you could add "dopamine" as a new type of "neurohormone".
  • Admin Panel for Conceputal Schema Mapping: While you could add "neurohormone" as a new type of feature, only Admins will be able to describe what a neurohormone is in relation to the other features. If you would like to be an Admin, please get in touch.
